Introducción

Este es un proyecto colaborativo de desarrollo web utilizando Git y GitHub. Aprenderemos a trabajar en equipo gestionando versiones de código.

Objetivo del Proyecto

El objetivo principal es comprender el flujo de trabajo en Git y GitHub, aplicando buenas prácticas de desarrollo colaborativo.

Recursos Utilizados

  • HTML5, CSS3 y JavaScript
  • Git para control de versiones
  • GitHub para la gestión de repositorios
